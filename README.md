# C++ Basic Calculator

A basic calculator that performs basic operations (+, -, *, /)

## How to Run

1. Clone this repository to your local machine.
2. Open the solution in Visual Studio (or your preferred C++ IDE).
3. Run the program.

## What It Does

- Performs basic math using these operations (+, -, *, /)
